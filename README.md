# React, Redux, Duck pattern and TS.

This is a react template using Redux, TS and Duck pattern.

## Getting started.

- Install modlues:

```
npm install
or
yarn install
```

- Run development:

```
npm run dev
or
yarn dev
```

[OPEN](http://localhost:3000)

- Generate Build for production;

```
npm run build
or
yarn build
```
